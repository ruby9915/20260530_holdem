# 용어집 — 홀덤 에이전트 연구 도메인

> 이 도메인 모든 트랙·문서의 **한국어 표기 단일 출처**(single source of truth).
> [`전역설정.md`](전역설정.md) §8(용어·문체 규약)이 이 문서를 참조한다. 표기가 헷갈리면
> 추측하지 말고 이 표를 따른다. 표에 없는 용어는 §0 원칙으로 판단하고, 판단 후 이 표에 추가한다.

## 0. 두 가지 표기 원칙 (가장 중요)

이 도메인은 성격이 정반대인 두 어휘군을 다루며, **표기 원칙이 서로 반대**다. 과거의 표기 불편(예: hand를 "손"으로 직역)은 이 둘을 구분하지 않아 생겼다.

1. **포커 도메인 용어 → 정착된 음차/원어를 쓰고 직역하지 않는다.**
   포커는 외래 게임이라 용어가 영어 기반으로 정착했다. `hand`를 "손", `pot`을 "냄비"로 직역하면 의미가 깨지고 비표준이다. 한국 포커 문헌·커뮤니티에서 통용되는 음차(핸드·팟·스택)를 그대로 쓴다.
2. **강화학습·통계·게임이론 용어 → 음차를 지양하고 한국 학계(KCI/KSCI) 정착 번역어를 쓴다.**
   `noise→잡음`, `baseline→기준선`처럼 학술 번역어가 있으면 그것을 쓴다.
3. **고유 알고리즘명·약어 → 원어 유지** (어느 군이든 공통). Monte Carlo, CFR, softmax, mbb/g, ZCA 등.

> **한 문장 요약**: *포커 용어는 음차 유지, 학술 용어는 번역, 알고리즘명·약어는 원어.*

## 2. 강화학습·통계 용어 — 학술 번역어 (음차 지양)

| 지양(음차) | 지향(학술어) | | 지양 | 지향 |
|---|---|---|---|---|
| 스킴(scheme) | **방식 / 기법** | | 노이즈 | **잡음** |
| 셋업(setup) | **설정 / 구성** | | 로버스트 | **강건한 / 강건성** |
| 케이스(case) | **경우 / 사례** | | 트레이드오프 | **절충 / 상충** |
| 리워드 | **보상** | | 베이스라인 | **기준선** |
| 컨버전스 | **수렴** | | 디스소시에이션 | **해리** |

**KCI 표준어 확정**(비표준·혼동 표기 → 학술 표준어):

| 지양(비표준/혼동) | 지향(학술 표준어) | 근거 |
|---|---|---|
| 진짜값 / 진짜 가치 | **참값 / 참 가치** | 통계·RL 학술어 |
| 기대값 | **기댓값** | 사이시옷 표준어(국립국어원) |
| 음성 결과 | **부정적 결과** | '음성'은 의학 진단 차용어 오해 |
| 보상 정형화 | **보상 형성**(reward shaping) | '정형화'=formalization 혼동 |
| optimistic init. | **낙관적 초기화** | 첫 등장 시 원어 병기 |

**유지 확정**(현행이 정확 — 바꾸지 말 것): 몬테카를로 **제어**(control), **통제실험/통제비교**(단일변수), **결정 상태**(MDP 상태; '결정 노드' 아님), 기여도 배분(credit assignment), 비례배분(proportional credit), 고정점·흡수·일반화·진단·사례연구·재현성·수렴. **greedy**는 원어 유지('탐욕적 정책' 아님).

## 3. 게임이론·CFR 용어 (트랙 02)

| 영어 | 표준 표기 | 비고 |
|---|---|---|
| CFR (Counterfactual Regret Minimization) | **CFR** | 약어 원어 유지. 풀이 시 "반사실적 후회 최소화" |
| counterfactual | **반사실적** | |
| regret | **후회 (regret)** | regret matching은 "후회 매칭" 또는 원어 |
| information set | **정보집합** | CFR 기본 단위 |
| exploitability | **착취가능성 (exploitability)** | best response 대비 손실. CFR 수렴 지표 |
| best response | **최적 대응** | |
| Nash equilibrium | **내쉬 균형** | |
| self-play | **자기대국 (self-play)** | |
| average strategy | **평균 전략** | CFR 산출물 |
| abstraction (card/action) | **추상화 (카드/행동)** | |
| node (game tree) | **노드** | 게임 트리 마디. ※ MDP의 '결정 상태'와 구분 |

> 게임이론 용어는 RL 군(§2)에 준해 **학술 번역어**를 쓰되, CFR·MCCFR 같은 약어·고유명은 원어를 유지한다(§0-3).

## 4. 약어·고유명

| 약어 | 풀이 |
|---|---|
| **mbb/g** | milli big blinds per game. payoff × 1000 / BB(=2) = payoff × 500. 주 평가 단위 |
| **OOD / ID** | Out-of-distribution(미학습 분포) / In-distribution(학습 분포 내). 역할은 학습 상대에 따라 뒤집힘 |
| **GTO** | Game Theory Optimal. 균형 전략 |
| **ZCA** (트랙 01) | Zero-Credit Absorption. 비용 0 행동의 0-credit이 argmax를 흡수하는 병리 |
| **VIC** (트랙 01) | Virtual Information Cost. CHECK에 1칩 가상 투자를 부여해 0-고정점 흡수를 해소 |
| **PROP / PURE** (트랙 01) | 비례배분 기여도(invest 비율×payoff) / 표준 MC(말단보상 γ-할인 역전파) |
| **MCCFR / CFR+** (트랙 02) | Monte Carlo CFR / regret matching+ 변형 |
| **페르소나** | 평가·학습 상대: TAG(타이트-공격), LAG(루즈-공격), Maniac, Station(콜링스테이션), Nit(초타이트) |

## 5. 단위·기호

| 기호 | 의미 |
|---|---|
| **mbb/g** | 주 성능 지표 (§4) |
| **SE** | 표준오차 = s / √N |
| **95% CI** | 신뢰구간 ≈ 평균 ± 2·SE |
| **α / γ / ε / τ** | 학습률 / 할인율 / 탐색 확률(ε-greedy) / 온도(softmax) |

## 부록. 표기를 정할 때

1. 포커 게임 자체의 개념인가? → §1 (음차 유지, 직역 금지).
2. 학습·통계·최적화 방법론인가? → §2/§3 (학술 번역어).
3. 알고리즘 고유명·약어인가? → 원어 유지.
4. 어느 쪽도 애매하면 동일 분야(강화학습·게임이론) 국내 KCI 논문 표기를 따르고, 정한 결과를 이 문서에 추가한다.

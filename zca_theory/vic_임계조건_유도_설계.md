# VIC 최적 비율 유도 — 세 설계안 심사 종합 및 실행 프로그램

## 결론 요약

- **주 경로 = C안(차원분석·무차원 붕괴)을 뼈대로, A안(고정점 유도)을 유도 엔진, B안(검정력 층)을 성공률 보정층으로 결합.** 세 안의 하한식은 표기만 다른 동일식이므로 "선택"의 실체는 논문 절 구성이다.
- **명칭 판정: "임계 조건 유도"** 채택. "임계식 유도" 지양, "최적 가상비용의 해석적 유도"는 상한 검증 전까지 과장.
- 실행 순서: 학습 0회 검증 4건(즉시) → 6단 55h 병행 작업(코드 훅·분석·경량 1런) → 신규 런은 저자 게이트 3건(15런 / 10런 / 조건부 10런).
- 사전 등록·판정 기준을 런 전에 실험일지에 명기 — 부정적 결과여도 논문이 되는 구조로 설계한다(3절).

## 1. 주 경로 판정과 이유

**세 안의 하한식은 동일식이다.** A안 (E3) `v_min = (M̂+c)/μ`, B안 (식 2) `ε_min = g/|ρ|`, C안 (식 2) `ε_min = k·c(s)/(1−k)`는 기호만 다른 같은 argmax 반전 조건이고, C안의 형태가 이미 `analyze_threshold.py:7`에 구현되어 있다. 따라서 유도부는 합의된 상태이며, 안별 고유 기여만 남는다.

| 안 | 고유 기여 | 논문 내 역할 |
|---|---|---|
| A (고정점·동역학) | 실제 갱신식(EMA·PROP 배분)에서의 유도 사슬, uniform_penalty·slowT 실패의 기전 설명(자기교정성), 양측 협공(희석) | **유도 절의 본문** — "왜 VIC만 되는가" |
| B (신호-잡음·검정력) | seed 성공률 = F에 Φ가 곱해진 층 — **F(5)=0.74 vs 관측 0.40 괴리의 유일한 정량 후보**, 학습/평가 2-층 분리 | 예측-관측 괴리의 통계 해명(보조 절 또는 부록) |
| C (차원분석·붕괴) | 무차원 좌표 ρ=K/ĉ 붕괴 법칙 — 상수 요동과 α 강건을 **한 식의 두 극한**으로 통일, 학습 0회 자기일관성 검사(P1), 마스킹 모드로 상한까지 같은 절차로 확장 | **검증 프로그램의 뼈대** — 주장·반증 구조 |

**KCI "유도 + 실증 검증" 절로 성립하려면 필요한 것** (이 결합이 유일하게 전부 충족):
1. 실코드 갱신식에서 출발한 유도 (A) — 사후 곡선적합이 아님을 보이는 근거.
2. 무차원 예측 법칙과 학습 0회 검증 (C의 P1) — 기존 데이터만으로 1차 반증 가능.
3. 관측-예측 크기 괴리의 해명 (B의 Φ층) — "방향 일치·크기 부분 설명"을 정량 서사로 승격.
4. **런 전에 수치로 사전 등록된 out-of-sample 예측** 최소 1건 (게이트 A/B) — 이것이 없으면 심사에서 "사후 설명"으로 격하된다.
5. 상한의 정직한 한정 — 현 앵커(K60 1점, 신규 코드에서 미재현)로는 상한식 주장 불가.

**논문 절 구성 권고**: ① 고정점 유도 → 임계 조건 → 무차원화 ② 검증 1: 기존 앵커 ~40점의 무차원 좌표 적합(학습 0회) ③ 검증 2: 사전 등록 예측 대조(신규 런) ④ 한계: 상한·온도 축·경로의존성.

## 2. 단계별 계획

### 0단계 — 기존 데이터로 즉시 (학습 0회)

1. **`analyze_threshold.py` 확장** (`c:\code\minimizing\poker-ladder\analyze_threshold.py`). 재독에서 확인한 선행 수정 사항 하나를 먼저 지적한다:
   - **L108–109가 masking·absorption 두 모드의 ε을 `all_eps` 한 분포에 합산하고 있다.** 하한(흡수 해소)과 상한(마스킹) 임계는 의미가 다르므로 분리 출력이 선행 조건이며, 기존 보고치(ε_min 중앙값 3.2/3.4/3.8칩)가 흡수 모드 순수치인지 재확인해야 한다. masking 비중이 작으면 기존 수치 유지, 크면 E9 수치 정정 기록.
   - 확장 항목: 팟 분위수·CDF 출력(L76 평균→분위수), α_min = ε_min/c(s) 분포 산출, 대상 플랫폼 k50_a12 추가(6단 완주 후 CFR 포함), 근거 Q 3종 교차(off-학습·fixed5-학습·chec_a30-학습 — C안 약점 1의 내생성 검사).
2. **P1 자기일관성 검사**: (a) med(α_min)이 α축 전이 중점(실측 8–15%)과 일치하는가, (b) 칩 단위 ε_min 중앙값은 플랫폼마다 다르되 α_min 중앙값은 플랫폼 간 근사 일치하는가, (c) α_min 분포의 8–20% 질량이 전이구간 seed 요동을, 25–30% 포화가 전 플랫폼 5/5+를 설명하는가.
3. **probit 동시 적합** (B안 P1): 기존 앵커점(K축 5점 + α축 6점 이상)을 무차원 좌표로 변환해 2-모수 이항 적합, 공통 폭 w 성립 여부. Φ-보정이 A12의 0.74→관측 0.40 괴리를 흡수하는지 정량 판정.
4. **사전 등록**: 게이트 A/B의 예측 수치(각 K·α의 기대 양수 seed 수)를 런 전에 `실험일지.md`에 기록. 판정 기준(어떤 결과면 기각인지)도 함께.

## 3. 핵심 위험 — 유도가 실측과 안 맞을 때의 해석 경로

| 실패 시나리오 | 해석 | 논문 격하 경로 |
|---|---|---|
| P1 실패 (α_min 중앙값이 플랫폼 간 불일치) | payoff 포화(스택 200 상계)·팟 분포 내생성으로 붕괴가 연성(soft)에 그침 | 주장을 "칩 단위 임계는 플랫폼 의존이며 팟-비례는 이를 1차 소거한다"로 한정 — 여전히 상수 요동 대 α 강건의 기전 설명으로 성립 |
| 게이트 A 실패 (K 순서 역전) | 정적 고정점 근사 기각 — 경로의존 동결값 M̂이 지배 | **"정적 임계 조건은 방향을 예측하나 크기는 경로의존" 자체가 기여** — 진단(ZCA)+처방(VIC)+임계 조건의 한계 실증으로 재구성. 부정적 결과 논문 가능하되 임팩트는 하락 |
| Φ층이 괴리를 못 흡수 | 잡음층이 원리적 유도가 아닌 통계 모형으로 전락 | 부록 격하, 본문은 F(v) 방향 예측만 주장 |
| 게이트 C 빈손 (하락 미관측) | 상한 채널 구성(희석·왜곡) 미확정 | 상한 절 삭제, 하한 정리만 — 사전 등록 덕에 "탐색 후 선택" 의심 회피 |

**최소 방어선** (이것만 성립해도 절이 산다): ① 실코드에서 유도된 하한 조건 ② 학습 0회 검증(P1) 부분 성립 ③ 사전 등록 예측 1건 이상 적중. 셋 다 실패하면 이론 절을 접고 32·33번 실측 서사로 회귀한다 — 이 판정도 사전 등록에 포함할 것.

**범위 조항 (필수 명기)**: 이 유도의 적용 범위는 흡수 해소(OOD-Random·수동 상대 회복)까지다. vsCFR 전 조건 음수, vsLAG/MAN 0/10은 흡수와 무관한 역량 문제 — α\*가 이를 개선한다는 예측은 하지 않는다.

## 4. 용어 판정

**채택: "임계 조건 유도" (critical condition derivation).** 근거:
1. 유도 대상이 단일 닫힌 등식이 아니라 **argmax 반전 부등식 조건과 그로부터의 임계값 분포**다. "임계식"은 단일 수식이라는 인상을 주어 과대 표현이고, 표준 학술 결합어도 아니다.
2. KCI 공학·물리 관례상 "임계 조건"(안정 임계 조건, 좌굴 임계 조건 등)이 정착 결합어다.
3. "최적 가상비용의 해석적 유도"는 상한이 사실상 무앵커인 현 상태에서 α\*의 존재·위치를 확정하는 인상을 준다 — 운영 원칙 5·7(정직성·보수적 사고) 위배 위험.

**절 제목 권고**: "가상 투자 비용의 임계 조건 유도와 실증 검증". 게이트 C 통과(상한 실측 확보) 시에만 "최적 가상 투자 비율의 해석적 근사"로 승격. 부수 표기: nondimensionalization = "무차원화"(정착 번역어), data collapse는 이 프로젝트가 이미 collapse를 "성능 붕괴"로 쓰므로 혼동 회피를 위해 "단일 곡선 중첩(collapse)"으로 구분하고 첫 등장 시 원어 병기 — `용어집.md` 추가 대상.

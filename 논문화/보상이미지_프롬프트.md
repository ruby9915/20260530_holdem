# 비례 배분 보상(Proportional Credit Assignment) 이미지 생성 프롬프트

> 이미지 생성 에이전트는 추상 개념을 잘 못 그립니다. 아래는 **구체 숫자 예시 + 화면 배치 + 그릴 요소**를 못 박은 프롬프트입니다.
> 가장 잘 통하는 것부터 순서대로 3가지 버전을 제공합니다. **버전 A(권장)** 부터 시도하세요.

## ✅ 버전 A — 한국어 인포그래픽 (권장, 그대로 붙여넣기)

```
포커 강화학습의 '비례 배분 보상'을 설명하는 교육용 인포그래픽 한 장을 그려줘.
스타일: 깔끔한 플랫 디자인, 네이비/파랑/주황/빨강 포인트 컬러, 흰 배경, 16:9 가로,
도형은 둥근 모서리 사각형, 화살표로 흐름 표시. 손글씨/사진 느낌 금지, 벡터 다이어그램 느낌.

[그림 구성 — 위에서 아래로 3개 영역]

(1) 상단 영역 제목: "한 핸드에서 내린 3번의 결정"
    아래에 막대 3개를 가로로 나란히. 막대 높이 = 베팅에 넣은 칩 수에 비례.
    - 파란 막대  : 라벨 "PREFLOP 레이즈", 값 "10칩"  (가장 낮음)
    - 주황 막대  : 라벨 "FLOP 베팅",    값 "20칩"  (중간)
    - 빨강 막대  : 라벨 "TURN 큰 베팅",  값 "60칩"  (가장 높음)
    오른쪽에 "합계 = 90칩" 표시.

(2) 중앙 영역: 큰 초록색 가로 띠 박스 하나.
    텍스트: "핸드 종료 → 순이익 payoff = +180칩 (승리)"

(3) 하단 영역 제목: "투입한 칩 비율 그대로 +180을 나눠 가진다"
    위 3개 막대에서 아래로 화살표가 각각 내려와, 같은 색의 결과 박스 3개로 연결:
    - 파란 박스 : "PREFLOP  +20"
    - 주황 박스 : "FLOP  +40"
    - 빨강 박스 : "TURN  +120"
    각 화살표 옆에 작은 분수 계산식:
       파랑 "10/90 × 180 = 20",  주황 "20/90 × 180 = 40",  빨강 "60/90 × 180 = 120"

[맨 아래 한 줄 강조 문구]
    "가장 크게 베팅한 TURN 결정이 +120으로 가장 큰 책임(보상)을 받는다"

핵심 메시지: 한 번의 핸드 결과(+180)를, 각 결정이 '베팅에 넣은 칩의 비율'대로
공정하게 나눠 갖는다는 것. 큰 베팅일수록 더 큰 보상/책임.
```

## 버전 B — 영어 (영어 모델에서 한글이 깨질 때)

```
Create one clean educational infographic explaining "proportional reward credit
assignment" in poker reinforcement learning. Flat vector style, white background,
16:9 landscape, rounded rectangles, navy/blue/orange/red accents, arrows for flow.
No photo, no handwriting.

TOP SECTION — title "Three decisions in one poker hand".
Three vertical bars side by side, bar height proportional to chips invested:
- Blue bar:   "PREFLOP raise" = 10 chips (shortest)
- Orange bar: "FLOP bet"     = 20 chips (medium)
- Red bar:    "TURN big bet" = 60 chips (tallest)
Right side label: "Total = 90 chips".

MIDDLE SECTION — one wide green band box:
"Hand ends -> net profit payoff = +180 chips (WIN)".

BOTTOM SECTION — title "Split +180 by the SAME ratio of chips invested".
Arrows go down from each top bar to a same-color result box:
- Blue box:   "PREFLOP  +20"   with small formula "10/90 x 180 = 20"
- Orange box: "FLOP  +40"      with small formula "20/90 x 180 = 40"
- Red box:    "TURN  +120"     with small formula "60/90 x 180 = 120"

Bottom caption: "The biggest bet (TURN) earns the biggest credit (+120)".

Core idea: one final hand result (+180) is divided among the decisions in proportion
to how many chips each decision put in the pot. Bigger bet -> bigger share.
```

## 버전 C — 비유 버전 (개념을 더 직관적으로)

투자 지분 비유를 쓰면 이미지 모델이 더 잘 이해합니다.

```
"공동 투자 후 수익 배분"을 포커에 빗댄 인포그래픽을 그려줘.
세 사람이 한 프로젝트에 각각 10만원, 20만원, 60만원(합 90만원)을 투자하고,
프로젝트가 180만원 수익을 냈을 때, 투자 비율대로 20/40/120만원을 나눠 받는 그림.

이걸 포커로 치환:
- 투자금 = 한 핸드에서 각 '결정'이 베팅에 넣은 칩 (10/20/60칩)
- 수익   = 그 핸드의 최종 순이익 (+180칩)
- 배분   = 투입 비율대로 각 결정에 돌아가는 보상 (+20 / +40 / +120)

좌측: 세 개의 투자 막대(파랑10·주황20·빨강60), 우측: 같은 색 수익 박스(+20·+40·+120),
가운데 큰 화살표와 "180을 비율대로 분배" 문구. 플랫 벡터 스타일, 16:9, 흰 배경.
```
